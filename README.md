# Hôtel de Luxe - Système de Réservation

## Description
Application web de réservation d'hôtel développée avec React et Node.js.

## Technologies
- Frontend : React.js
- Backend : Node.js / Express.js
- Base de données : MySQL
- Authentification : JWT

- Aya - Reservations & Home Page
